## Design System

> Referências visuais: portfólio de engenheiro com sidebar lateral + hero com vídeo fullscreen
> Minimalismo editorial — hierarquia tipográfica, cor vem das imagens, fundo sempre neutro.

### Paleta de Cores

#### Modo claro (default)

| Token CSS | Valor | Uso |
|---|---|---|
| `--bg` | `#fafaf9` | Fundo da página (off-white quente) |
| `--surface` | `#ffffff` | Cards, superfícies elevadas |
| `--surface-dim` | `#f4f4f2` | Fundo de seções alternadas, CTA block |
| `--text-primary` | `#0d0d0c` | Títulos, corpo principal |
| `--text-secondary` | `#525250` | Textos de apoio, parágrafos |
| `--text-muted` | `#9a9a96` | Labels, meta, nav links, períodos |
| `--border` | `#e8e8e4` | Divisórias, grid de cards |

#### Modo escuro

| Token CSS | Valor |
|---|---|
| `--bg` | `#0b0d13` |
| `--surface` | `#12141d` |
| `--surface-dim` | `#16181f` |
| `--text-primary` | `#e2e0db` |
| `--text-secondary` | `#7a7c88` |
| `--text-muted` | `#484a56` |
| `--border` | `#1c1e2c` |

**Cor de destaque:** nenhuma accent color fixa. As imagens/vídeos trazem a cor — fundo sempre neutro. Botão primário = dark fill `#0d0d0c` (não usa cor).

### Tipografia

- **DM Sans** — única família; cobre todos os pesos (400 regular · 500 medium · 600 semibold · 700 bold)
- Sem fonte mono separada — simplicidade e coesão visual

#### Escala tipográfica

| Token | Tamanho | Uso |
|---|---|---|
| `--fs-xs` | `11px` | Labels, eyebrows, tags, meta |
| `--fs-sm` | `13px` | Nav, corpo secundário, botões |
| `--fs-md` | `15px` | Corpo principal |
| `--fs-lg` | `18px` | Sub-headings, títulos de card |
| `--fs-xl` | `24px` | — |
| `--fs-2xl` | `36px` | Section titles, números de métricas |
| Hero title | `clamp(34px, 4.5vw, 54px)` | H1 do hero |

- letter-spacing heroes: `-0.025em`
- letter-spacing eyebrows: `0.12em` + `text-transform: uppercase`
- line-height corpo: `1.65–1.7`

### Espaçamento — escala 8-grid

| Token | Valor | Referência |
|---|---|---|
| `--sp-1` | `8px` | gaps mínimos |
| `--sp-2` | `16px` | gaps internos |
| `--sp-3` | `24px` | padding de cards |
| `--sp-4` | `32px` | margens de bloco, hero margin-top |
| `--sp-5` | `48px` | padding de seções compactas (metrics) |
| `--sp-6` | `64px` | padding padrão de seções |
| `--sp-7` | `72px` | padding base de página (CTA bottom) |

**Regra:** usar sempre valores do grid — nunca valores hardcoded arbitrários.

### Layout

- **Container:** `max-width: 960px` · padding lateral `48px` (`--sp-5`)
- **Body:** `padding-top: 88px` — compensa a dock nav fixa no topo
- **Hero:** dentro do container, `border-radius: 16px`, `height: 520px`
- **Separadores de seção:** `border-bottom: 1px solid var(--border)` — sem blocos de cor

### Border Radius

| Token | Valor | Uso |
|---|---|---|
| `--radius-sm` | `8px` | grids de cards (services, projects) |
| `--radius-md` | `12px` | eco cards, CTA block |
| `--radius-lg` | `16px` | hero |
| Botões pill | `100px` | CTAs e botões de ação |

### Componentes

**Hero com vídeo**
- `<video autoplay muted loop playsinline>` como fundo (`object-fit: cover`)
- Overlay: `linear-gradient(to right, rgba black 0.88 → 0.1)` — texto à esquerda, imagem respira à direita
- Conteúdo: eyebrow + H1 + body + 2 botões pill, alinhados ao bottom-left via `flex-end`

**Grid de serviços (3×2)**
- `gap: 1px; background: var(--border)` — cria divisórias sem CSS extra
- Cada célula: `background: var(--surface)`, `padding: 24px`, icon + title + desc

**Eco Cards (4 colunas)**
- Fundo escuro com `linear-gradient` temático por categoria
- Overlay `to top` com fade
- Conteúdo: icon + title + desc + arrow circular `24px`
- Hover: `translateY(-2px)` apenas

**Grid de Projetos (5 colunas)**
- Mesmo padrão `gap: 1px; background: var(--border)`
- Nome da empresa em tipografia variada (peso/tamanho diferente por cliente para identidade)
- Hover: `background: var(--surface-dim)`

**Métricas (row horizontal)**
- 5 itens separados por `border-right: 1px solid var(--border)`
- icon (16px muted) + número grande (`--fs-2xl`, peso 700) + label (`--fs-xs`, muted)

### Interações & Animações

- Hover em links: `transition: color 0.15s` ou `opacity 0.15s`
- Hover em eco cards: `transform: translateY(-2px)` · `transition: 0.2s`
- **Sem** animações de entrada (scroll-triggered), **sem** parallax, **sem** JS de efeito visual
